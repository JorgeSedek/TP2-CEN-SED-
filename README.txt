Decidimos hacer que nuestra implementación solo funcione con coordenadas mayores  a cero.
Las filas y columnas comienzan a contarse desde 1.

Es decir, si en el archivo "ubicaciones.txt" por ejemplo, si hay una coordenada del tipo
(0, 1) ó (1, 0) el programa no correrá correctamente.

Esto lo hicimos por una cuestión de tener una interfaz más agradable para el usuario.
